Bank Management System

Bank Management System is a web-based application built using Java that simulates how a bank operates. It features three login portals for customers, employees, and admin. Customers can register, deposit, and withdraw money with proper validation. Employees can log in with their details, and the admin can view profiles of both customers and employees. The system also includes monthly interest calculation and balance updates.
